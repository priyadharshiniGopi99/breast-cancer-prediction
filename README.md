# 🧠 Breast Cancer Prediction System

This project is a Machine Learning-based web application that predicts whether a tumor is **Benign or Malignant** using multiple classification algorithms.

---

## 🚀 Features

* Predict breast cancer using input values
* Uses multiple ML algorithms:

  * Logistic Regression
  * Decision Tree
  * Random Forest
  * SVM
  * KNN
* Displays **best performing algorithm**
* Web interface using **FastAPI + HTML/CSS**

---

## 🛠 Tech Stack

* Python
* Scikit-learn
* FastAPI
* HTML, CSS

---

## 📊 Dataset

* Breast Cancer Wisconsin Dataset
* 30 features used for prediction

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```

Open:

```
http://127.0.0.1:8000
```

---

## 📌 Project Workflow

1. Data preprocessing
2. Model training (5 algorithms)
3. Model comparison
4. Best model selection
5. Deployment using FastAPI
